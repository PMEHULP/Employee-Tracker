# Employee-Tracker
Interactive dashboard to track employee distribution, attrition, and salary trends across departments. Includes KPI cards, filters (Gender, Status, Quarter), and clear visuals to provide actionable insights and improve workforce monitoring.
📊 Employee Tracker Dashboard (Power BI)
🧠 Project Overview

The Employee Tracker Dashboard is an interactive Power BI solution designed to monitor and analyze employee data across departments. It provides a clear overview of workforce distribution, attrition trends, and salary insights to support data-driven decision-making.

🎯 Objectives
Track total employee count and attrition metrics
Analyze department-wise employee distribution
Monitor salary trends across departments
Provide interactive filtering for deeper analysis
📌 Key Features
KPI Cards: Total Employees, Total Attrition, Attrition %
Department Analysis: Employee count by department
Designation Breakdown: Role-wise distribution using donut chart
Salary Insights: Department-wise salary trends
Interactive Filters: Quarter, Gender, and Employee Status
Detailed Table: Department and designation-level data
📊 Insights Delivered
Identify departments with highest and lowest employee count
Understand attrition patterns across workforce
Compare salary distribution between departments
Analyze employee composition by designation
🛠️ Tools & Technologies
Power BI Desktop
Data Modeling & DAX
Data Visualization & Dashboard Design
🚀 Outcome

This dashboard enables efficient employee tracking, improves visibility into workforce trends, and supports better organizational planning through intuitive and interactive visuals.
